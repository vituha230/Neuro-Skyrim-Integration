# Neuro-Skyrim-Integration

I have developed an integration that can let Neuro play Skyrim (solo playthrough).

Currently the entire main quest can be completed.

The integration is ready to be tested.. Untested run is probably possible too (but I'd recommend testing since I have no idea how actual neuro works and was using some substitute llms).

Tested using Gary app (with engines gemma3-12b and qwen3-14b).

Mechanics that are supported by the integration:
- walking
- fighting (melee and ranged)
- using magic
- ability to follow a quest
- opening doors and interacting with things (like pulling levers)
- dialogues
- barter
- crafting (alchemy, smithing, enchanting, gear upgrading)
- context info about surrounding events and objects
- using items from inventory (and changing equipment)
- ability to use map (both fast-travel and normal walking to location)
- collecting berries and other ingredients
- catching butterflies
- doing crimes
- escaping prisons
- levelup skill and attribute choice
- reading books
- spin
- fishing
- lockpicking

Some demonstration videos:
- [Cave raid](https://www.youtube.com/watch?v=kQVDKba-8-8)
- [Main quest dungeon](https://www.youtube.com/watch?v=QBQCrqnrAjM)

## Requirements
* **Skyrim Special Edition 1.6.1170** latest steam release
  * [Optional] Launch at least once to main menu (populates registry so tooling can discover path automatically)
  * Note: **Skyrim Anniversary Edition 1.6.1170** should also work, but is untested
* [XMake](https://xmake.io) [2.8.2+]
  * `powershell -NoProfile -ExecutionPolicy Bypass -Command "iex (irm https://xmake.io/psget.text)"`
* C++23 Compiler ([MSVC](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-0-installation?view=msvc-170), [Clang-CL](https://clang.llvm.org/get_started.html))
  * Easiest to install via [winget](https://github.com/microsoft/winget-cli), [scoop](https://scoop.sh/), or [choco](https://chocolatey.org/install)
  * MSVC - `winget install Microsoft.VisualStudio.2022.Community`, then install [c++ workload](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-0-installation?view=msvc-170#step-4---choose-workloads)
  * Clang-CL (released as part of [LLVM](https://llvm.org/)) - `winget install --id=LLVM.LLVM`

## Getting Started
### Clone Repository
```bat
git clone --recurse-submodules https://github.com/vituha230/Neuro-Skyrim-Integration
cd Neuro-Skyrim-Integration
```

### [Setup](docs/Setup.md)
```bat
xmake first-time-setup
```

### [Testing](docs/Testing.md)
You'll need to run Gary in order to drive the Neuro Integration.

### (Re)Launch
```bat
xmake launch
```

### Tasks
To view all available tasks, just run `xmake tasks`. To view details for a specific task: `xmake [task] --help`
