# Neuro-Skyrim-Integration

Skyrim mod that can allow an LLM to play the game using text inputs


Some demonstration videos:
- [(OLD) Cave raid](https://www.youtube.com/watch?v=kQVDKba-8-8)
- [(OLD) Main quest dungeon](https://www.youtube.com/watch?v=QBQCrqnrAjM)
- [Random Riften](https://www.youtube.com/watch?v=p20JZZP4EyQ)
- [Random Riften 2](https://www.youtube.com/watch?v=MOQIoj-e-90)
- [Long walk](https://www.youtube.com/watch?v=IKCNaaQnnBw)

## Requirements
* **Skyrim Special Edition 1.6.1170** (latest Steam release)
  * [Optional] Launch at least once to main menu (populates registry so tooling can discover path automatically)
  * Note: **Skyrim Anniversary Edition 1.6.1170** should also work (was not tested)
* [XMake](https://xmake.io) [2.8.2+]
  * `powershell -NoProfile -ExecutionPolicy Bypass -Command "iex (irm https://xmake.io/psget.text)"` (or install it manually)
* C++23 Compiler ([MSVC](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-0-installation?view=msvc-170), [Clang-CL](https://clang.llvm.org/get_started.html))
  * Easiest to install via [winget](https://github.com/microsoft/winget-cli), [scoop](https://scoop.sh/), or [choco](https://chocolatey.org/install)
  * MSVC - `winget install Microsoft.VisualStudio.2022.Community`, then install [c++ workload](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-0-installation?view=msvc-170#step-4---choose-workloads)
  * Clang-CL (released as part of [LLVM](https://llvm.org/)) - `winget install --id=LLVM.LLVM`

## Getting Started
### Clone Repository
```bat
git clone --recurse-submodules https://github.com/vituha230/Neuro-Skyrim-Integration
cd Neuro-Skyrim-Integration
```

## Setup
Quick one-line setup:
```bat
xmake first-time-setup
```
Alternatively:
### [Setup](docs/Setup.md) - quick guide
### [Detailed Setup](docs/DetailedSetup.md) - detailed guide

## Testing/Running
You'll need to use neuro-api-compatible engine ([Gary](https://github.com/Govorunb/gary)/[Jippity](https://github.com/EnterpriseScratchDev/neuro-api-jippity)/[Tony](https://github.com/Pasu4/neuro-api-tony)/[Randy](https://github.com/VedalAI/neuro-sdk/tree/main/Randy)/etc) in order to drive the Neuro Integration.
### [Testing](docs/Testing.md) - testing guide


### Launching the game
Launching through xmake:
```bat
xmake launch
```
Manual launch:
using `skse64_loader.exe` from the game folder

### Tasks
To view all available tasks, just run `xmake tasks`. To view details for a specific task: `xmake [task] --help`
